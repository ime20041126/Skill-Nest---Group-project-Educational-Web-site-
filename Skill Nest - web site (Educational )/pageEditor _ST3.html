Digital Literacy Page

Technical Description

  • Five section layout using semantic HTML5 tags
  • Visuals with alt tags and consistent styling
  • Sticky “Go to Top” button using fixed positioning

Accessibility

  • Alt text for all images
  • Readable text and responsive layout
  • Accessible navigation with clear anchor links

View Digital Literacy Page Validation

Challenges and Lessons Learned

  • Challenge: Making the layout responsive across devices → Solved using media queries and grid/flex
  • Challenge: JavaScript-based popup confirmation → Solved by testing with different events and reset logic
  • Lesson: Design consistency and accessibility should be planned from the start

Compliance

  • Pages comply with W3C HTML5 and CSS3 standards
  • Accessibility implemented using WCAG 2.1 guidelines
  • Navigation and layout structure follow Janet web publication requirements
